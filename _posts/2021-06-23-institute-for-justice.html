Project for the Institute for Justice

Rationale

The homepage of the Institute for Justice does a great job of drawing users in to IJ’s important work. You immediately see faces and get the message: “This place is helping real people!”

But once you start to navigate the page, the user experience gets a little clunky. The “Support IJ,” “Give Now,” and “Report Abuse” buttons all compete for your attention. And when you hover your mouse over these buttons, you get surprising responses—“Give Now” turns dark and the hamburger menu suddenly flashes a new color scheme.

Smoothing out these little bumps in the UX is not just a vanity project. Thoughtful design increases the likelihood of users engaging with calls to action (e.g. the “Give Now” button). Implementing this redesign could increase donations.

Proposal

I created a new navigation bar. By giving it a dark but translucent color, the logo stands out more, but you still see the background image. The user’s eye is drawn to the “Give Now” button by a pop of color, but they can still see the “Report Abuse” and “Support IJ” buttons clearly.

Then, instead of changing colors or adding borders around icons, the site can respond to user’s hover action with an elegant animation of the button or icon.

You can view all the code used for these changes below. Just a few lines of CSS can help people use the site more easily.

{% highlight css %} /* Adding a dark, transluscent nav background */ .re-menu-container { background-color: rgb(0, 0, 0, 0.6); } /* Adjusting colors and hover animations on menu items */ .re-menu-box { transition: all 200ms ease-in-out } .re-menu-box:hover { background-color: transparent; color: white; transform: translateY(-5px) } .re-menu-search { background-color: transparent; border: 0px solid black; transition: all 200ms ease-in-out; } .re-menu-search:hover { transform: translateY(-5px) } .re-menu-donate { background-color: #10757b; border: 3px solid #10757b; transition: all 200ms ease-in-out; letter-spacing: 2px; } .re-menu-donate:hover { transform: translateY(-5px); } .re-menu-report-abuse { background-color: transparent; border: 0px solid black; transition: all 200ms ease-in-out; letter-spacing: 2px; } .re-menu-report-abuse:hover { transform: translateY(-5px); } /* Changing colors on the “Support IJ” button */ /* (This is not currently a class on the website) */ .support-ij { background-color: #000000; color: #FFFFFF; border-right: 1px solid #393939 } {% endhighlight %}

Conclusion

I believe these changes can provide real value for the Institute for Justice and have the potential to increase funding. And while these suggestions are related to the front-facing user experience, they demonstrate the conscientious problem-solving that I apply across the web development stack.
